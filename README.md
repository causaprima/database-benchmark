# Homework 4. Databases (Postgres and MongoDB) benchmark

## Features
* GraphQL (/graphql)
* Postgres
* MongoDB
* Databases 'write' and 'read' benchmark

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running the benchmark

```bash
# databases benchmark
$ npm run test:databases
```
